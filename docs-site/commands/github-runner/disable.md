# azlin github-runner disable

Disable GitHub Actions runner fleet.

## Synopsis

```bash
azlin github-runner disable --pool <pool-name>
```

## Examples

```bash
azlin github-runner disable --pool ci-workers
```

Stops all runners and removes runner registration.
