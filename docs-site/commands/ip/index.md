# IP Commands

IP diagnostics and network troubleshooting.

## Commands

- [check](check.md) - Check IP address classification and connectivity

## Quick Start

```bash
# Check single VM
azlin ip check my-vm

# Check all VMs
azlin ip check --all
```

## Use Cases

- Diagnose connectivity issues
- Verify public vs private IP assignment
- Troubleshoot SSH connection problems
- Validate Bastion configuration
